# BMR Calculator (Flutter)

## Anggota Kelompok:
- Zanjabiela Adil Perkasa (K3523079)
- Ilham Muhammad Hakiem (K3523037)

## Deskripsi
Aplikasi ini menghitung kebutuhan kalori basal (BMR) berdasarkan:
- Gender
- Tinggi badan
- Berat badan
- Umur

BMR dihitung menggunakan rumus Harris-Benedict.
